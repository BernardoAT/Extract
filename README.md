Bash function Extract
=======

Bash function for extract: .zip, .rar, .bz2, .gz, .tar, .tbz2, .tgz, .Z, .7z, .xz, .ex, .tar.bz2, .tar.gz, .tar.xz


## How to install (Mac OS X)

Copy/Paste function into file `~/.bash_profile`
